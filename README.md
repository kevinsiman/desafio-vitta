Todo List feito para o Desafio Vitta

A aplicação tem como função criar uma lista de tarefas com:

Nome da Tarefa
Descrição da Tarefa
Campo de Tags
Data de Termino da Tarefa
Hora de termino da tarefa


Funções:
Filtrar tarefas por: 

periodo de tempo
tarefas concluidas

Organizar tarefas por Nome ou data

Adicionar Tarefas
Remover Tarefas
Atualizar Tarefas
